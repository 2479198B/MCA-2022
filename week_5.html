Metadata Schema

Below is a description of the metadata elements I have added to the MEI file.

  • File elements:
    • Title - the name of the piece that the score represents: Air in D Major
    • Composer - the original creator of the piece: Johann Sebastian Bach
    • File size: 94.8 kB
    • Year - the year of file creation: 2022
    • Source - name of source: IMSLP
    • Source link - link to original files: https://imslp.org/wiki/Special:ReverseLookup/254645
  • Encoding elements:
    • Encoder name: Sophie Barcan
    • Year - the year of encoding: 2022
  • Work elements:
    • Notes on broader work - context, if the score is part of a collection, an album etc.: From Suite No. 3 in D major, BWV 1068
    • Date - original publication or creation year: 1730
